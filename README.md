# CleverenceSoft

Проект состоит из трёх заданий, реализованных на C#:

1. Компрессия и декомпрессия строки
2. Сервер для чтения и записи
3. Обработка и стандартизация лог-файлов

Также присутствуют unit-тесты на NUnit.

---

## Задача 1: Компрессия и декомпрессия строки

**Условие:**
Дана строка из маленьких латинских букв.
Требуется заменить группы повторяющихся символов на формат "sc", где s - символ, c - количество повторений.
Если символ встречается один раз - число не указывается.

**Пример:**

Вход:  aaabbcccdde
Выход: a3b2c3d2e

**Реализовано:**
- Compression() - сжатие строки
- Decompression() - восстановление исходной строки

---

## Задача 2: Многопоточный сервер

**Условие:**
Есть статический класс-сервер с полем count и двумя методами:
- GetCount() - чтение значения
- AddToCount(int value) - увеличение значения

К серверу одновременно обращаются много читателей и несколько писателей.

**Требования:**
- Читатели работают параллельно, не блокируя друг друга
- Писатели работают последовательно (эксклюзивно)
- Во время записи читатели ждут

## Задача 3: Стандартизация лог-файлов
**Условие:**
Программа читает лог-файл с записями в двух разных форматах и приводит их к единому стандартному виду.

**Входные форматы:**

**Формат 1:**
10.03.2025 15:14:49.523 INFORMATION Версия программы: '3.4.0.48729'

**Формат 2:**
2025-03-10 15:14:51.5882| INFO|11|MobileComputer.GetDeviceId| Код устройства: '@MINDEO-M40-D-410244015546'

**Выходной формат:**

Дата | Время | Уровень | ВызвавшийМетод | Сообщение

Дата: формат DD-MM-YYYY

Время: сохраняется в исходном формате

Уровень: INFORMATION -> INFO, WARNING -> WARN

Если метод отсутствует - подставляется DEFAULT

Невалидные записи попадают в problems.txt

**Пример вывода:**

2025-03-10 15:14:49.523 INFO DEFAULT Версия программы: '3.4.0.48729'
2025-03-10 15:14:51.5882 INFO MobileComputer.GetDeviceId Код устройства: '@MINDEO-M40-D-410244015546'

**LogParser** - распознавание форматов 1 и 2
